# Meu Portfolio

Este repositório contém um portfólio pessoal com várias sessões de exercícios.

## Estrutura

- `index.html` — página inicial (root) com links para todas as sessões
- `sessao2/`, `sessao3/`, `sessao4/` — cada pasta tem a sua própria `index.html` para as respetivas sessões

As páginas estão pensadas para serem publicadas através do **GitHub Pages**; basta ativar o Pages a partir da branch `main` (ou `master`) e os links funcionam de forma relativa.

## Navegação

Cada `index.html` possui um menu de navegação com ligações para:

1. Seções internas (`Sobre Mim`, `Projetos`, `Contacto`)
2. Outras sessões do portfólio (Sessão 2, Sessão 3, Sessão 4)
3. Um link de regresso à página inicial quando se está numa pasta de sessão

Os caminhos foram configurados de forma a funcionarem mesmo no ambiente de GitHub Pages.
